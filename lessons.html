Lesson 2: Algorithms

An algorithm is a step-by-step procedure to solve a problem.

Example:
Algorithm to add two numbers:
1. Start
2. Input A, B
3. Sum = A + B
4. Output Sum
5. Stop

Lesson 4: Pseudocode

Pseudocode is a simplified way of writing programming logic without syntax.

Example:
BEGIN
  READ A, B
  SET C = A + B
  PRINT C
END
